
  # ValidateNow UI

  Frontend for the ValidateNow admin portal.

## Running the code

Run `npm install` to install the dependencies.

Run `npm run dev` to start the development server.

The dev server port is read from `VITE_FRONTEND_PORT` in `.env`. The default is `5175`.

The backend API URL is derived from the current browser host and `VITE_API_PORT`. The default API port is `8005`. Use `VITE_API_BASE_URL` when you need to override the full backend URL.
